# Whack-a-Mole Game

A fun browser based **Whack-a-Mole** game built using **HTML, CSS, and JavaScript**.  
The game features multiple difficulty levels, a timed mode, sound effects, animations, and separate high scores per difficulty.

---

## Features

- **Multiple difficulty levels:** each affecting the spawn speeds and number of plants
- **Timed Mode:** Race against the clock (60 secs)
- **Classic Mode:** Keep playing until you hit a plant or your score drops to negative
- **Visual Effects:** on making a new highscore and when the timer is about to run out in timed mode
- **Sound Effects:** on correct and wrong hits and also a timer warning

---

## Gameplay

- Click on the mole to **gain points**
- Avoid the piranha plants — clicking one ends the game
- Clicking an empty tile **reduces your score**
- If your score drops below zero, it’s **game over**
- Watch out for the timer in the timed mode
